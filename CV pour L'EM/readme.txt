Pour l'utiliser rien de plus simple il suffit de cliquer sur le fichier "page_principal.html" ou de l'ouvrir avec un navigateur web (les navigateurs qui fonctionnent sont "Google chrome",

effets soit le fluide et les image les plus claires.
